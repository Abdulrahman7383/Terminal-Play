# 🎮 Terminal-Play

مرحباً بك في مستودع لعبة **Terminal-Play**! هي لعبة كلاسيكية تم تطويرها بالكامل باستخدام لغة **C++**. تم تصميم اللعبة لتقدم تجربة تفاعلية ممتعة ومباشرة وتعمل بالكامل داخل بيئة موجه الأوامر (Console/Terminal).

---

## ✨ المميزات (Features)
* 🕹️ **لعب عبر الشاشة السوداء (Console-based):** تجربة لعب كلاسيكية خفيفة وسريعة لا تتطلب أي مكتبات رسومية خارجية.
* 🏗️ **برمجة كائنية التوجه (OOP):** تم بناء اللعبة باستخدام هياكل برمجية نظيفة وقابلة للتطوير.
* ⏱️ **استجابة فورية:** تحكم سلس ومباشر لمدخلات لوحة المفاتيح داخل بيئة الـ Terminal.
* 📊 **نظام نقاط (Score System):** حفظ وحساب النقاط الحالية أثناء اللعب.

## 🛠️ المتطلبات (Requirements)
لتشغيل اللعبة وبنائها، ستحتاج فقط إلى:
* مترجم لغة C++ يدعم معيار **C++11** أو أحدث (مثل `GCC` أو `Clang` أو `MSVC`).
* محرر أكواد أو بيئة تشغيل تدعم تشغيل الأوامر (Terminal / Command Prompt / IDE Console).

## 🚀 كيفية بناء وتشغيل اللعبة (How to Run)

اتبع الخطوات التالية لتشغيل اللعبة محلياً:

### 1. استنساخ المستودع (Clone the Repository)
```bash
git clone https://github.com/YOUR_USERNAME/Terminal-Play.git
cd Terminal-Play
```

### 2. بناء المشروع (Compilation)
إذا كنت تستخدم مترجم GCC (`g++`)، يمكنك كتابة الأمر التالي في موجه الأوامر:
```bash
g++ -std=c++11 main.cpp -o TerminalPlay
```

### 3. تشغيل اللعبة (Execution)
* **على أنظمة Linux / macOS:**
  ```bash
  ./TerminalPlay
  ```
* **على نظام Windows:**
  ```cmd
  TerminalPlay.exe
  ```

## 🎮 أزرار التحكم (Controls)
* 🔼 **W / سهم للأعلى:** التحرك للأعلى.
* 🔽 **S / سهم للأسفل:** التحرك للأسفل.
* ◀️ **A / سهم لليسار:** التحرك لليسار.
* ▶️ **D / سهم لليمين:** التحرك لليمين.
* ⌴ **Space (المسافة):** إطلاق النار / التفاعل.
* ❌ **Esc / Q:** الخروج من اللعبة.
